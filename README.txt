**Conversational Q&A Chatbot using Streamlit and LangChain**

---

## Overview

This repository contains a conversational Q&A chatbot implemented using Streamlit, OpenAI, and LangChain. The chatbot allows users to ask questions and receive responses generated by the OpenAI language model. The conversation history is displayed in a user-friendly interface using Streamlit.

---

## Setup

### Prerequisites

Make sure you have the following installed:

- Python 3.x
- Pip (Python package installer)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo.git
   ```

2. Navigate to the project directory:

   ```bash
   cd your-repo
   ```

3. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

---

## Usage

1. **Environment Setup:**

   - Ensure you have set up a virtual environment for the project.

     ```bash
     python -m venv venv
     ```

   - Activate the virtual environment:

     - On Windows:

       ```bash
       .\venv\Scripts\activate
       ```

     - On Unix or MacOS:

       ```bash
       source venv/bin/activate
       ```

2. **Run the Streamlit App:**

   ```bash
   streamlit run app.py
   ```

   This will launch the Streamlit app in your default web browser.

3. **Interacting with the Chatbot:**

   - Enter your questions in the provided input box.
   - Click the "Ask the question" button to submit your question.
   - The chatbot will provide responses generated by the OpenAI language model.
   - The conversation history will be displayed on the interface.

---

## Files and Directory Structure

- **app.py:** Main Streamlit application script.
- **requirements.txt:** List of Python dependencies.
- **.env:** Environment variables (Make sure to include your OpenAI API key here).
- **/venv:** Virtual environment folder (automatically generated).

---

## Important Notes

- Make sure to include your OpenAI API key in the `.env` file for the chatbot to function properly.
